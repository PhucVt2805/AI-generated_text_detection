[PYTHON] AI-generated Text Detection

Introduction:


Here is my python source code for the problem of checking if a text is AI generated or not. With my code:

  • You can check if the entire text is AI-generated.
  • You can check each sentence if it is AI generated or not.
  • You can fine-tuning one (or more) models yourself to suit your usage needs.
